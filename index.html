Possuo habilidades em HTML, CSS, SASS, JavaScript, TyperScript e Git. Tenho conhecimento em tecnologias como REST e AJAX, consigo integrar serviços web de maneira eficiente, utilizando a abordagem assíncrona do AJAX para melhorar a interatividade e integrando APIs RESTful para uma comunicação eficaz entre sistemas. Minha adaptabilidade e paixão pelo desenvolvimento web me capacitam a abordar desafios de codificação de maneira eficaz. Estou em busca de oportunidades para aplicar meu conhecimento e crescer na área, contribuindo para projetos desafiadores que impulsionem minha evolução como Engenheiro Front-End.

Meus Projetos

Webpage

O Clone Disney+ é uma reprodução simplificada da interface da plataforma de streaming Disney+, oferecendo uma experiência interativa de navegação e visualização de conteúdo.

Gerenciador de tarefas

Aplicativo de gerenciamento de tarefas construído usando Vue.js. Ele permite que você adicione novas tarefas, marque tarefas como concluídas e filtre as tarefas com base em seu status de conclusão.

Calculadora de IMC

A Calculadora de IMC é uma aplicação em ReactJS que permite aos usuários calcular rapidamente seu Índice de Massa Corporal (IMC), uma medida de saúde que avalia a relação entre altura e peso. Ideal para monitorar a saúde e adotar hábitos mais saudáveis.
